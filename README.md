# Overview
Bun? Deno? Node.js? Creating your own JavaScript Runtime using V8, Libuv and more. dudeJS is a JavaScript runtime built on top of [V8 Javascript Engine](https://github.com/v8/v8) and [Libuv](https://github.com/libuv/libuv). This repo is about recreating the mighty Node.js from Scratch using V8, Libuv and C++
